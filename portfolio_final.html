프로젝트 개요

해양 펩타이드 및 독성유전자 데이터를 통합 구축하고 시각화·고급 검색 기능을 제공하는 생명정보 플랫폼. AI 기반 벡터 유사도 검색(FAISS)을 도입하여 기존 키워드 검색의 한계를 넘어서고, MongoDB로 비정형 생명정보 데이터를 효과적으로 관리합니다.

담당 업무
  • 해양 펩타이드 통합정보 구축 및 시각화 기능 개발 (Angular + Spring Boot)
  • 독성유전자 통합정보 구축 및 시각화 기능 개발
  • ETL 파이프라인 설계 및 구축 (정제 → MongoDB 저장)
  • CLI 기반 대량 데이터 등록 기능 구현 (배치 처리)
  • MongoDB 기반 DB 구축 (스키마 설계 포함)
  • AI 기반 유사도 검색 구현 — 벡터화 + FAISS 인덱스
  • 다국어(i18n) 서비스 개발 (한국어 / 영어)
  • 일괄 등록 서브 기능 개발
